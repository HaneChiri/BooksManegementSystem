# LibraryManegementSystem
# 图书管理系统

一个学习使用面向对象方法编程的项目

要实现的功能：

- 添加、删除、修改、查询图书信息
- 用户（读者、管理员）注册，登陆图书馆系统
- 添加、删除、修改、查询账号信息
- 记录借阅及归还日志
- 记录管理员操作日志
